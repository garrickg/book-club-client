Book Club

The Book Club allows users to share their book collections with others. Browse the available books, find a book you like, and send the owner a request to borrow!

This project was completed as part of the Free Code Camp Dynamic Web Applications Curriculum. The user stories for this project are:

  • I can view all books posted by every user.
  • I can add a new book.
  • I can update my settings to store my full name, city, and state.
  • I can propose a trade and wait for the other user to accept the trade.
